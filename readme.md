# Adaptive Network and Coupled Dynamics Simulations

## Overview
This repository provides a comprehensive suite of Python scripts designed to simulate the co-evolution of adaptive networks and dynamic hygiene contagion. The models investigate how socio-physical systems respond to various interventions, tracking metrics such as information state, systemic damage, and economic phase transitions.

## Core Modules

* **`main1.py`**: Conducts sensitivity analysis on non-linear distortion (theta) and propagation coefficients (p) using coupled ordinary differential equations.
* **`main2.py`**: Evaluates unstable elements by testing varying recovery rates (mu) and intervention timings (t_int) to observe changes in economic phase transitions.
* **`main3.py`**: Simulates targeted (hub-centric) versus random network interventions on a Barabasi-Albert scale-free network to measure mitigation efficacy.
* **`main4.py` & `main5.py`**: Implements full adaptive network co-evolution, enabling nodes to dynamically rewire connections and propagate structural resilience (hygiene contagion) away from contaminated sources.
* **`main6.py`**: Compares the effectiveness of localized structural bypasses (dynamic edge deletion) against global algorithmic circuit breakers triggered by high infection thresholds.

## Visualizations
Each module automatically exports structured CSV data alongside high-resolution analytical plots. Key visual outputs generated by the scripts include:
* Sensitivity comparison plots mapping efficiency phase transitions.
* Targeted versus random network intervention trajectories.
* Adaptive network structural decay and intervention overlays.

## Installation and Setup
Clone the repository to your local machine or server environment:

```bash
git clone [https://github.com/RUDATAScience/adaptive-network-simulations.git](https://github.com/RUDATAScience/adaptive-network-simulations.git)
cd adaptive-network-simulations

```

Install the required dependencies. The core computational logic relies on standard scientific libraries:

```bash
pip install numpy pandas scipy networkx matplotlib

```

## Execution

Run any simulation script from the terminal. For example, to test the structural bypass and circuit breaker dynamics:

```bash
python main6.py

```

Upon execution, the scripts calculate the time steps, compile the resulting data into an output directory, generate the corresponding `matplotlib` graphs, and compress the results into a downloadable `.zip` archive.
